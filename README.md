MinecraftHelper
Prosty Minecraft Helper dla Windows, który ułatwia PVP oraz kopanie tuneli za pomocą konfigurowalnych makr i komend.
​

Tutaj możesz wkleić 1–2 screeny z aplikacji (np. główne okno + zakładka Kopacz).

Funkcje
Czytelny GUI

Górny pasek statusu z informacjami o zapisaniu ustawień.
​

Panel z nazwą okna gry i kafelkami stanu trybów (LPM+PPM, Auto LPM/PPM, Kopacz 5/3/3, 6/3/3).
​

Panel statusu gry

Wspólny dla wszystkich zakładek pasek „Minecraft Focus / Settings Saved”.
​

Kolorowe kropki pokazujące, czy Minecraft ma fokus i czy konfiguracja jest zapisana.
​

Tryb PVP

Makro LPM + PPM HOLD z osobnymi CPS dla lewego i prawego przycisku myszy.
​

Osobne tryby AUTO LPM i AUTO PPM z własnymi bindami i zakresem CPS.
​

Tryb Kopacz

Profil Kopacz 5/3/3 – lista komend wykonywanych z opóźnieniami pod jednym klawiszem.
​

Profil Kopacz 6/3/3 – kopanie „na wprost” lub „do góry” z konfiguracją szerokości/długości.
​

Wbudowany „przelicznik czasu” (minuty → sekundy) pomocny przy ustawianiu komend.
​

Ustawienia i historia tytułów

Pole „Tytuł okna gry” oraz lista ostatnio używanych tytułów (max 5) z przyciskiem „Ustaw”.
​

Dzięki temu można szybko przełączać się między różnymi serwerami/launcherami.
​

Import / Export konfiguracji

Zapisywanie wszystkich ustawień do pliku json oraz ich wczytywanie przyciskiem z poziomu GUI.
​

Automatyczny zapis do settings.json w %AppData%\MinecraftHelper.
​

Jak uruchomić
Sklonuj repozytorium:

bash
git clone https://github.com/SzybkiPoPiwo/MinecraftHelper.git
Otwórz rozwiązanie w Visual Studio 2022 (lub nowszym) z zainstalowanym .NET dla WPF.
​

Ustaw projekt MinecraftHelper jako startowy i uruchom.
​

Krótka instrukcja
W zakładce Ustawienia wpisz tytuł okna Minecrafta (np. „Minecraft” albo dokładną nazwę okna) i zapisz.
​

W zakładce PVP ustaw CPS oraz klawisze dla trybów, których chcesz używać.
​

W zakładce Kopacz skonfiguruj komendy i bindy dla 5/3/3 oraz 6/3/3.
​

Upewnij się, że kropka Minecraft Focus jest zielona – wtedy makra będą działać na okno gry.
